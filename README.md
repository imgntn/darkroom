## Local Development

1. Install dependencies and build the bundle:
   ```bash
   npm install
   npm run build
   ```

2. Start the server:
   ```bash
   npm start
   ```

Then open [http://localhost:3333](http://localhost:3333) in your browser.

Use the **Mute** button (or press **M**) to toggle audio. The preference
persists across sessions.
